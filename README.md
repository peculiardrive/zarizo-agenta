# 🚀 ZARIZO: Agent-First Digital Commerce Platform

**Zarizo** is a full-stack, production-ready ecosystem designed to automate sales through a powerful network of agents. Built using **Next.js 14**, **Supabase**, and **Tailwind CSS**, it empowers businesses to scale their reach with minimal manual intervention.

---

## 🌟 Key Features

### 🏢 Business Dashboard
- **Product Management**: List inventory with custom pricing and flexible commission structures.
- **Order Pipeline**: Real-time tracking of sales from commission to delivery.
- **Agent Analytics**: Monitor top-performing agents and payout history.

### 🎓 Agent Dashboard
- **Sharable Links**: Generate unique referral links for any store product.
- **Live Commissions**: Real-time tracking of pending and approved earnings.
- **Payout System**: Direct bank account integration for seamless earnings withdrawal.

### 🛡️ Admin Master Panel
- **Global Overview**: Manage all platform businesses, agents, and transactions.
- **Automation Oversight**: Monitor Edge Functions for order processing and commission approvals.
- **User Verification**: Robust KYC and approval workflows for businesses and agents.

---

## 🛠️ Technical Stack

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Database / Auth**: [Supabase](https://supabase.com/)
- **Payments**: [Paystack](https://paystack.com/)
- **UI & Styling**: [Tailwind CSS](https://tailwindcss.com/), [Framer Motion](https://www.framer.com/motion/)
- **Messaging**: [Termii](https://termii.com/), [Resend](https://resend.com/)

## 🚀 Installation & Local Environment

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/zarizo.git
    cd zarizo
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Environment Variables**:
    Create a `.env.local` file and fill in your Supabase and Paystack credentials (see the provided `.env.local.example`).

4.  **Launch the App**:
    ```bash
    npm run dev
    ```
